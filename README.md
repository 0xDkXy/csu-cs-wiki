## 关于项目

本项目开源了部分 CS 专业的课程实验与课程设计的源码,我们同时还架设了一个 Wiki 网站，我们的核心想法是：

- 为代码开源做点贡献
- 通过 Wiki 中对知识点的扩展，帮助大家更好地了解学到的内容是如何在实际项目中运用
- 利用 Wiki 帮助同学们更好地理解或复习课程的**核心知识点**
- 加速学院一些课程实验与课程设计的**代码迭代**（部分实验用的是 10-15 年前的代码，太过时了）

如果你喜欢，别忘了给我们一个 Star💫！
