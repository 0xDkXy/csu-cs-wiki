## 概要
上过的课的课程实验与课设代码开源，其目的为：  
 1. 让学校课程使用了十几年的实验内容进行更新
 2. 帮助大家通过代码更好地进行学习
 
目前已包括：
 1. 计算机图形学
 2. 密码学
 3. 大型数据库技术
 4. Linux
 5. 数字图像处理
 6. 汇编
 7. 分布式与云计算（未完成）
 8. [JAVA](https://github.com/leo6033/Java_Project)
 9. [动画与游戏程序设计](https://github.com/leo6033/UnityLearing/tree/master/TanksWar)

有用的话别忘了点个 star 

## 参与
欢迎大家参与该项目，参与的方式大概如下：
 + 该项目中还未涉及到的课程
 + 该项目中未包括的实验
 + 对已有代码进行优化
